# getting-rusty 🦀

A personal repo documenting my journey learning Rust — mostly through DSA and small exercises.

## What's in here

- **dsa/** — data structures & algorithms
- **mini_project/** — try to do mini project
- **introduction/** — early experiments and syntax exploration

## Why Rust

Coming from C and Java, wanted to explore a language that takes
memory safety seriously without a garbage collector.

## Notes

This repo is for learning purposes, not production code.
